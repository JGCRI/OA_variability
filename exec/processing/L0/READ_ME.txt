This is a read me file for the following directory, \...\exec\processing\L0

This folder contains the two csv input required to run the subsequent levels of R processing for the Ocean Acidification project.
	- the defined_basins.csv contains the lat/lon definitions of the ocean basins used to define the basin area for basin means
		- L0_basin_boundary.R and basins_maps.Rmd are files associated with the visualizing the basin boundaries. 
	- the models_to_remove.csv contains a list of the models and the reasons why we decided to exclude these models from future analysis. 
		note: this file and code that uses this file might not want to be made public.
